Bem-vindo ao Animu

Nossa Missão

O Animu é uma plataforma dedicada aos fãs de anime, criada com o objetivo de fornecer informações detalhadas e organizadas sobre as mais diversas séries de animação japonesa.

Nossa missão é criar um ambiente acolhedor onde os fãs possam descobrir, explorar e compartilhar sua paixão por animes.

Recursos

  • Banco de dados extensivo de animes
  • Informações detalhadas sobre cada série
  • Sistema de classificação e avaliação
  • Interface intuitiva e responsiva
  • Suporte a temas claro e escuro
  • Atualizações regulares

Tecnologias

HTML5
CSS3
JavaScript
JQuery
Firebase
Tailwind CSS
Jikan API
Swiper
Flaticon
Quill.js

Equipe

O Animu é desenvolvido por estudantes apaixonados por programação e anime, focados em criar a melhor experiência possível para nossa comunidade.
